108204019 尤敏米茲夠

1. 遊戲操作

WASD移動角色、E鍵攻擊

2. 主要遊戲機制

貓咪要擊退敵人保護島嶼中央的農場，藉由擊退敵人取得分數，且存活得越久越好。

4. 特色或額外功能

每波敵人會從四面生成進攻，貓咪要擊退敵人保護島嶼中央的農場；
前4波每波40秒、第5波到第9波有60秒、第10波以上有90秒，每波敵人數呈線性上升，依不同種類敵人的權重隨機生成；
前4波只有基礎敵人（刺蝟、松鼠、犰狳、狐狸）、第5波到第7波出現響尾蛇、第8波以上出現蜘蛛；
基礎敵人要打1~2下才會死、響尾蛇3下、蜘蛛4下，基礎敵人擊殺得100或200分、響尾蛇400分、蜘蛛800分；
角色有攻擊、跑步、閒置動畫、敵人有加上受到傷害、死亡和碰到農場的動畫；
敵人有AI自動生成路徑。
